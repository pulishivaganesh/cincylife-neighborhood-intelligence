# CincyLife Neighborhood Intelligence

## Problem

People moving to Cincinnati often struggle to choose neighborhoods that fit their budget, safety requirements, transportation needs, and lifestyle preferences.

## Goal

Build a data-driven dashboard that helps users compare Cincinnati neighborhoods.

## Metrics

- Rent Affordability
- Crime Rate
- Parks
- Walkability
- Public Transportation
- Population Density

## Technology Stack

- Python
- Pandas
- SQL
- Git
- GitHub
- Power BI
- VS Code

## Future Enhancements

- AI-powered recommendations
- Interactive maps
- Neighborhood ranking system
